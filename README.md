## 🔧 快速开始

```bash
# 安装依赖
bundle install

# 本地预览
bundle exec jekyll serve

# 访问 http://localhost:4000
```

## 📁 重要文件

- `_config.yaml` - 网站配置（标题、描述、链接）
- `index.md` - 主页内容
- `_members/*.md` - 成员信息
- `_data/highlights.yaml` - 论文列表
- `images/headshots/` - 成员头像（**注意：需要替换为真实照片**）
