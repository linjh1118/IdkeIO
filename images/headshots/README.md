# 成员头像目录

## 说明

这个目录用于存放所有成员的头像照片。

## 如何添加头像

1. 准备头像图片：
   - 建议尺寸：400x400 像素（正方形）
   - 格式：JPG 或 PNG
   - 文件大小：建议不超过 500KB

2. 文件命名规则：
   - 使用成员文件名对应的图片名
   - 例如：`_members/bo-sun.md` 对应 `images/headshots/bo_sun.jpg`
   - 注意：横线 `-` 在文件名中变成下划线 `_`

3. 上传图片：
   ```bash
   # 将图片复制到这个目录
   cp your-photo.jpg images/headshots/member-name.jpg

   # 提交并推送
   git add images/headshots/
   git commit -m "添加成员头像"
   git push
   ```

## 临时解决方案

在你添加真实头像之前，网站会显示默认的占位符图片。
